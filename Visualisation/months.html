In this map the months range between February 2016 and August 2017. The month can be selected in the control panel. The map is split up into blue grids of 10x10 degrees. For the duration of the month the meetings in the grid were counted. The number of meetings is displayed on the marker which is clickable for further information. The marker also provides the number of days this month consisted of in the data, as some months were missing data, which can give an unjust representation. You would think that the map would showcase the differences in meetings over time, but it is more realistic to say that it shows the difference in data over time, as some months were missing days and some days consisted of a lot more messages.
